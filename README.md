## Overview
This repository contains the development of **Task 05 (TA05)**. The project focuses on building a structured web application using modern frontend standards. It demonstrates the ability to organize content with HTML5, apply visual layouts with CSS3, and implement client-side interactivity using JavaScript.

## Repository Structure
The project is organized into the following directory structure to maintain clean and scalable code:

* **`index.html`**: The main entry point of the website.
* **`/HTML`**: Contains additional sub-pages and site sections.
* **`/CSS`**: Stylesheets for visual design, typography, and responsive layouts.
* **`/JS`**: JavaScript files for dynamic functionality and user interaction.
* **`/Images`**: Graphic assets and multimedia used throughout the site.
* **`/Projectes`**: Specific sub-projects or exercises related to this assignment.
* **`/.idea`**: Development environment configuration files (WebStorm/IntelliJ).

## Technologies Used
* **HTML5**: For semantic content structuring.
* **CSS3**: For custom styling and layout management.
* **JavaScript (ES6+)**: For implementing interactive elements.

    Navigate to the project folder and open `index.html` in any modern web browser (Chrome, Firefox, Safari, or Edge).
